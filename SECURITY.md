# Security Policy

We take security issues seriously. If you discover a vulnerability, please report it responsibly so we can address it before disclosure.

---

## 🔒 Reporting a Vulnerability

**DO NOT** open public GitHub issues for security vulnerabilities. Instead, report them directly:

1. Send an email to **tanmaysingh8246@gmail.com** with the subject line: `Security Vulnerability: <Brief Description>`.
2. Include a detailed description of the vulnerability, reproduction steps, proof of concept (PoC), and potential impacts.
3. We will acknowledge receipt of your report within 48 hours and work on a fix.
4. We request that you keep the details confidential until we publish a patch.
