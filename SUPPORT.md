# Support Guide

If you need help or have questions regarding the Whisper Sandbox, use the following resources:

---

## 🙋 GitHub Discussions & Issues

- **Bug Reports**: If you find an issue in the code, search existing issues first. If it's a new bug, file a report using the Bug Report template.
- **Feature Requests**: If you want to suggest an improvement or new feature, open a Feature Request issue.
- **General Questions**: If you need setup help or want to discuss design patterns, please check the `/docs` or reach out via email.
